# basic-page-template

Template for starting a standalone html page

Comes with

- HTML boilerplate `index.html` file with resources linked.
- CSS `reset.css` and `styles.css` files.
- JS `script.js` file.
- Favicon set up with a default.
- 3 free-to-use images.
